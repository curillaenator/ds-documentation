## Basic color

Основные цвета в X.Y.Z для набора стилей определеяющих наполненость компонентов и фирменного стиля

## Effect

Тональный переход через черный и белый слой, cтандартное значение в дизайн системе X.Y.Z имеет шаг прозрачности в 4%, но может варироваться при необходимости создания своего шага в определении тонального перехода под проект

## Sample

Компонент содержащий слои, которые накладываются поверх основного цвета, и дает возможность настроить тональный спект основного цвета

## Sample combination

Совмещенная палитра, базовых цветов и наложение эффектов, для выбора цветовых оттенков
