Пакет экспортирует 2 пресета - простой Avatar и AvatarWithThumb, который отображает в себе миниатюру.

Общие для пресетов props:

  • person - имя пользователя
  • subline - дополнительная подпись
  • src - ссылка на картинку
  • initials - инициалы в аватаре, когда нет ссылки на картинку. Будет отображено не более 2х символов
  • title - атрибут заголовка для аватара
  • disabled - отображение аватара как неактивный

Так же согласно макету AvatarWithThumb может быть только формы rectangular и размера large, поэтому у этого пресета пропс size и shape недоступны
